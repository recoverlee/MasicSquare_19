## TC-MS-A-001

| 항목 | 내용 |
|------|------|
| **테스트 ID** | TC-MS-A-001 |
| **테스트 일자** | *(실행일 기입)* |
| **테스트 목적** | 4×4 격자에서 빈칸(`0`)이 정확히 2개일 때, 행·열 우선 순서로 두 좌표를 올바르게 반환하는지 검증한다. |
| **테스트 기능** | `find_blank_coords` |
| **입력값** | `0`이 두 칸인 4×4 `int` 격자 (예: 명세서에 적은 두 빈칸 배치). |
| **테스트 단계** | **Given** 빈칸 2개인 입력 격자가 준비된다.<br>**When** `find_blank_coords`(또는 동등 API)를 호출한다.<br>**Then** 반환 리스트는 길이 2이며, 좌표가 행·열 오름차순과 일치한다(프로젝트 규약에 따라 1-index면 `[(r1,c1), (r2,c2)]` 형태로 기대값을 명시한다). |
